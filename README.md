# BYOKey Lab Site

BYOKey Labの公式サイトと、BYOKey Speak for Englishの製品ページです。

## Routes

- `/` - BYOKey Lab
- `/speak/english/` - BYOKey Speak for English
- `/guide/api/` - API設定ガイド
- `/privacy/` - プライバシーポリシー

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Cloudflareへのデプロイや設定変更は、ユーザー確認後に行います。
